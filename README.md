# Shopify AI Product Content Generator

A lightweight AI-powered tool for generating **SEO-optimised Shopify product content** such as titles, descriptions, bullet benefits, meta descriptions, tags, and image alt text.

Built to demonstrate a **realistic agency-style approach** to integrating AI into ecommerce workflows while keeping API keys secure and maintaining clean separation between frontend and backend.
